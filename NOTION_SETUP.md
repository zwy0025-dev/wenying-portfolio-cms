# Notion 数据库设置指南

## 1. 创建 Notion Integration

1. 访问 https://www.notion.so/my-integrations
2. 点击 **"New integration"**
3. 填写信息：
   - Name: `Portfolio CMS`
   - Associated workspace: 选择你的工作区
4. 点击 **"Submit"**
5. 复制 **Token**（以 `ntn_` 或 `secret_` 开头）

## 2. 创建数据库

在你的 Notion 工作区中，创建4个数据库：

---

### 数据库 1: 工作经历

**数据库标题**: `工作经历`

**属性设置**:
| 属性名 | 类型 | 说明 |
|--------|------|------|
| **时间段** | 标题 | 时间段，如 "2025.10 - 至今" |
| 职位 | 文本 | 职位名称，如 "餐厅合伙人/企业顾问" |
| 公司 | 文本 | 公司名称，如 "和牛定食餐厅/予童科技" |
| 描述 | 文本 | 职责描述 |
| 链接 | 链接 | 详情链接（可选） |
| 排序 | 数字 | 排序序号，数字越小越靠前 |

---

### 数据库 2: 项目经历

**数据库标题**: `项目经历`

**属性设置**:
| 属性名 | 类型 | 说明 |
|--------|------|------|
| **标签** | 标题 | 标签，如 "系统调优" |
| 项目名称 | 文本 | 项目名称 |
| 描述 | 文本 | 项目描述 |
| 链接 | 链接 | 详情链接（可选） |
| 排序 | 数字 | 排序序号 |

---

### 数据库 3: AI实验室

**数据库标题**: `AI实验室`

**属性设置**:
| 属性名 | 类型 | 说明 |
|--------|------|------|
| **分类** | 标题 | 类型，如 "AI Agent" |
| 项目名称 | 文本 | 项目名称 |
| 描述 | 文本 | 项目描述 |
| 链接 | 链接 | 详情链接（可选） |
| 媒体 | 文件 | 图片或视频（可选） |
| 排序 | 数字 | 排序序号 |

**媒体上传说明**:
- 支持格式：图片（jpg、png、gif、webp）、视频（mp4、webm、mov）
- 点击「媒体」列的空白处，选择「上传文件」
- 每个项目只能上传一个媒体文件
- 视频会自动显示播放控件

---

### 数据库 4: 核心技能

**数据库标题**: `核心技能`

**属性设置**:
| 属性名 | 类型 | 说明 |
|--------|------|------|
| **技能名称** | 标题 | 技能名称，如 "产品策略" |
| 排序 | 数字 | 排序序号 |

---

## 3. 连接 Integration 到数据库

对每个数据库执行以下操作：

1. 打开数据库页面
2. 点击右上角的 **"..."**（更多）
3. 选择 **"Add connections"**（添加连接）
4. 搜索并选择你创建的 Integration（`Portfolio CMS`）
5. 点击 **"Confirm"**（确认）

## 4. 获取数据库 ID

1. 打开数据库页面
2. 复制浏览器地址栏中的 URL
3. URL 格式：`https://www.notion.so/workspace/[32位ID]?v=...`
4. 提取 32 位的数据库 ID

示例：
```
URL: https://www.notion.so/myworkspace/1a2b3c4d5e6f7g8h9i0j1k2l3m4n5o6p?v=...
数据库 ID: 1a2b3c4d5e6f7g8h9i0j1k2l3m4n5o6p
```

你的数据库 ID：
- 工作经历：`32f9e034180c807883f0cbecf21698c6`
- 项目经历：`32f9e034180c80ce8504eeab4656e6e3`
- AI实验室：`32f9e034180c8079a1c9ee850d855318`
- 核心技能：`32f9e034180c803b928cfed839f917bd`

## 5. 配置环境变量

将获取到的信息填入 `.env.local` 文件：

```bash
NOTION_TOKEN=ntn_你的Token
NOTION_WORK_DB_ID=32f9e034180c807883f0cbecf21698c6
NOTION_PROJECT_DB_ID=32f9e034180c80ce8504eeab4656e6e3
NOTION_AI_LAB_DB_ID=32f9e034180c8079a1c9ee850d855318
NOTION_SKILLS_DB_ID=32f9e034180c803b928cfed839f917bd
```

## 6. 添加数据

现在你可以在 Notion 中像填写表格一样添加、编辑、删除内容了！

网站会自动读取这些数据并展示。

---

## 使用技巧

- **排序**: 修改 `排序` 字段的数字来调整显示顺序（1最靠前）
- **隐藏**: 暂时不想显示的内容可以删除或移到其他页面
- **链接**: 可以添加外部链接（如项目详情页、Notion 页面等）
- **格式化**: 描述支持普通文本，重要内容可以用 **粗体** 标记

## 数据同步

- 本地开发：刷新页面即可看到最新数据
- Vercel 部署：数据会自动缓存 60 秒，之后会自动刷新
